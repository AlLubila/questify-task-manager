# Questify Frontend

React SPA built with Vite and served by a small Node server.

## Requirements
- Node.js 18+

## Setup
1. Install dependencies:

```bash
npm install
```

2. Run the app in dev mode:

```bash
npm run dev
```

3. Build and run the production server:

```bash
npm run build
npm run start
```

## Environment
- `VITE_API_URL` (API base URL)
- `PORT` (frontend server port for `npm run start`)
